# Галерея робіт

Кидай роботи сюди — вони автоматично з'являться в секції «Галерея» на сайті.
Поки папка порожня, показуються тимчасові зразкові фото.

## Як називати файли

Усі файли лежать прямо в цій папці (без підпапок) і починаються з номера,
який задає порядок показу:

- `<n>-before.<ext>` — **оригінал** (стара / чорно-біла). Обов'язково.
- `<n>-animated.mp4` — **анімований ролик**. Якщо є — саме він показується
  з другого боку шторки.
- `<n>-after.<ext>` — відреставроване фото. Використовується з другого боку,
  **якщо для цього номера немає** `-animated`.

`<ext>` для фото: `jpg`, `png`, `webp`, `avif`. Відео — `mp4`.

## Приклад

```
public/gallery/
  1-before.jpg
  1-animated.mp4      ← шторка покаже відео
  2-before.jpg
  2-after.jpg         ← відео немає → шторка покаже це фото
  3-before.png
  3-animated.mp4
```

## Важливо

Сайт читає цю папку **під час збірки**. Після додавання робіт закоміть зміни
і передеплой (Vercel пересоберe сайт автоматично).
